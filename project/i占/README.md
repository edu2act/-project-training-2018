# 2018级项目实训成果展示 

## 《i占》 -  HTML5与移动互联网开发方向
###  项目简介

> i占——让教室占用更简单

i占项目旨在解决传统线下申请教室占用的费时费力问题，实现信息化管理教室。摆脱了纸质化提交申请材料以及教务管理老师与学生之间信息交互不及时的问题，使学院学生以及老师更便捷的查看教室使用情况。同时还提供了学院活动通知、最新公告以及故障报修等模块，真正实现“智慧软件学院“。

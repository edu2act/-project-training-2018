# 2018级项目实训成果展示 

## UI自动化测试、接口测试平台、移动端测试、性能测试 -  学无止境小组 - 软件测试方向

###  项目简介

> 软件测试

完成了对米拓建站系统的整体测试，实现了持续集成;搭建了接口测试平台，并使用其测试接口;对移动端健身助手ki进行了测试;最后对Discuz论坛进行了性能测试。

# 2018级项目实训成果展示 

## 《影·行》 -  Java与移动智能设备开发方向

###  项目简介

> 影·行——和电影一起去旅行

该项目为热爱电影及旅行人士提供便利、贴心的旅游生活服务，为其提供路线，并根据其自身情况规划路线，提供动态分享及直播，使用户在追寻自己独特电影回忆的同时遇到自己想看的风景。解决了用户在电影中出现的场景无法查到，来回进行旅游打卡路线规划较为复杂，没有志同道合的人进行沟通体验感差的问题。

# 2018级项目实训成果展示 

## 《简礼》 -  HTML5与移动互联网开发方向

###  项目简介

> 简礼——让你的婚礼不再手忙脚乱

一款专门为备婚一族提供良好选择服务的APP，旨在解决如今结婚行业存在的同质化、模式传统等问题。通过一系列关于婚纱摄影、婚礼策划以及婚纱礼服的推荐，既可以解决时间紧张的问题，又可以满足用户个性化的需求，给用户带来一场不留遗憾的婚礼。

# 2018级项目实训成果展示 

## 《嘻游记》 -  Java与移动智能设备开发方向

###  项目简介

> 嘻游记——让旅行更轻松

嘻游记是一款旅行分享类的App，传统的旅行攻略分享结合线上旅行功能，让用户在家实现云旅行，旨在提高用户的旅行体验感，找到性价比最高的出行路线。
